# LLM-Model-Requirements-Calculator
Микросервисная система для оценки совместимости LLM с вашим ПК Парсит название модели, автоматически собирает характеристики системы (RAM, VRAM, GPU) и даёт рекомендации: запустится ли модель локально, какое квантование использовать, или лучше обратиться к облачному API.
